# Инструкция по установке и запуску TelegramNode

## Быстрая установка

### 1. Клонирование репозитория
```bash
git clone https://github.com/oiv-an/TelegramNode.git
cd TelegramNode
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Настройка конфигурации
```bash
cp .env.example .env
# Отредактируйте .env файл с вашими данными
```

### 4. Запуск
```bash
npm start
```

## Подробная установка

### Системные требования
- **Node.js** 18+ (скачать с https://nodejs.org/)
- **npm** (входит в состав Node.js)
- **Git** для клонирования репозитория

### Получение Telegram API credentials
1. Перейдите на https://my.telegram.org/
2. Войдите в свой аккаунт Telegram
3. Перейдите в "API development tools"
4. Создайте новое приложение
5. Сохраните API ID и API Hash

### Зависимости проекта

Основные зависимости:
- `telegram` - клиент Telegram API
- `express` - веб-сервер для API
- `dotenv` - управление переменными окружения
- `form-data` - для отправки файлов

### Первоначальный запуск
При первом запуске потребуется:
1. Ввести код подтверждения из Telegram
2. При необходимости ввести пароль 2FA

### Проверка работы
- API сервер будет доступен на порту 3000 (или указанном в SERVER_PORT)
- Проверить статус: http://localhost:3000/health
- API для пересылки: POST http://localhost:3000/forward

### Возможные проблемы

**Ошибка "API_ID/AP_HASH_INVALID"**
- Проверьте правильность API ID и API Hash в .env файле

**Ошибка "PHONE_NUMBER_INVALID"**
- Убедитесь, что номер телефона указан в международном формате (+7...)

**Проблемы с загрузкой медиа**
- Убедитесь, что у вас есть права на скачивание медиа из канала
- Проверьте, что канал не ограничивает скачивание контента
